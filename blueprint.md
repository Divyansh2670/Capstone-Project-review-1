# **App Name**: Zenith CRM

## Core Features:

- Contact Management: Add, update, delete, and search contacts.
- Task/Appointment Management: Create and manage tasks/appointments linked to contacts.
- Dashboard Overview: Display total contacts, pending tasks, and upcoming tasks.
- User Authentication: Secure user registration and login using JWT.
- API Endpoints: Provide RESTful API endpoints for contacts and tasks.
- Data Persistence: Store and retrieve data using Prisma ORM with SQLite for development.
- Intelligent Contact Summarization: AI-powered summarization tool for contact notes, providing key insights at a glance. The summarizer will focus on identifying the 'next steps'.

## Style Guidelines:

- Primary color: Golden yellow (#F9A400) for a sense of energy and optimism.
- Background color: Light gray (#E2E2E2) for a clean and modern feel.
- Accent color: Dark gray (#333333) for contrast and emphasis.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look, suitable for both headlines and body text.
- Minimalistic icons to represent contacts and tasks.
- Whitespace-heavy layout with rounded cards and subtle shadows.
- Smooth fade & slide animations for transitions and interactions.
